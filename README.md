# KK_StudioVoice
コイカツのスタジオ内でボイスを流すプラグインです。  
このプラグインは、KK_StudioMoanにインスパイアされ作成したものです。先駆者様の活動に感謝致します。  
また、開発はAIアシスタントのサポートを受けながら行いました。  
私自身はプログラミングの専門家ではないため、予期せぬ不具合が発生する可能性があります。  
## 導入方法
`KK_StudioVoice.zip` をダウンロードし解凍したものをそのまま`BepInEx\plugins\` フォルダの中に配置してください。
## 機能説明
`Koikatsu\abdata\sound\data\pcm\`にある音声データをキャラクターの性格及び同梱している`KK_StudioVoice.csv`と照らし合わせて流す音声を決定する仕組みになっています。  
音声データを差し替えたり、csvを書き換えることで流す音声を変更することも可能です。
### スタジオでの動作方法
スタジオ左端のアイコンをクリックするとウィンドウが開きます。  
ワークスペースでキャラクターを選択し、各種プルダウンから音声を選択します。  
タイムライン上でキーフレームを入力し、流す音声を時間経過で切り替えることもできます。  
| 項目名 | 説明 | 例 |
| :--- | :--- | :--- |
| Anime Mode | 音声データの大枠となるカテゴリーです。 | 手コキ・正常位など |
| Anime Clip | Anime Modeの中で分類されたカテゴリーです。 | 弱ループ・絶頂など |
| Option | Anime Clipから更に細分化されたカテゴリーです。特にない場合はNoneとなります。 | Clip:キスの中のディープキスなど |
| Exp | キャラクターの性経験を切り替えます。コイカツ本編と同じく処女・不慣れ・慣れ・淫乱の4項目に分かれます。 |  |
| Excited | キャラクターの興奮状態のON/OFFを切り替えます。同カテゴリー内でボイスの強弱を切り替える際に使用します。 |  |
| LipSync | リップシンク同期のON/OFFを切り替えます。ONでもタイムライン上で制御されている場合は同期しません。 |  |
| Voice | セリフが流れるかどうかのON/OFFを切り替えます。 |  |
| Breath | 息遣いが流れるかON/OFFを切り替えます。初期設定ではセリフのない喘ぎ声などはこちらに分類されています。 |  |
| Play | 押すと音声再生が開始されます。もう一度押すと音声を停止します。 |  |
### csvの構成
先述したとおり、同梱のcsvを書き換えることで流す音声を自分好みに変更することができます。  
また、スタジオUI上のMode,Clipなどのカテゴリー名を変更することもできます。  
ただし、csvの項目を入れ替えたり追加することはできません。  
| 項目名 | 説明 |
| :--- | :--- |
| Anime Mode | 音声データの大枠を決定します。 |
| Voice Fallback | ダークネスの音声など一部の性格にしか存在しないボイスの代替先を記載します。空白の場合は代替されません。 |
| Breath Fallback | 同じく、代替先のブレスを記載します。Voice Fallbackと異なるカテゴリーでも問題ありません。 |
| Anime Clip | Modeの中で分類されたカテゴリーを記載します。 |
| Option | Clipの中で更に細分化されたカテゴリーを記載します。特にない場合は空白で問題ありません。 |
| Excited | 空白で興奮OFF、"1"で興奮ONの時に流れる音声になります。 |
| Not Loop | "1"で音声がループしないようになります。また、再生が終わるまで他の音声で上書きされない状態になります。 |
| Voice Prefix | `Koikatsu\abdata\sound\data\pcm\`にあるアセット番号の始め部分を指します。ボイス用です。 |
| Voice ID | こちらは後半部分を指します。Prefix + コイカツの性格番号 + ID で流れるボイスを指定しています。 |
| Breath Prefix  | ブレス用のPrefixです。 |
| Breath ID | ブレス用のIDです。 |
| Memo | メモを残したい場合はこちらに記載して下さい。音声読み取りに影響はありません。 |
### 細かい仕様
* **流れる音声の優先度について**
  * 基本的にボイスの再生終了後にブレスが流れます。
  * ボイスはループせず、ブレスはループするため、同じカテゴリーを流しっぱなしにした場合は「ボイス → ブレス → ブレス...」のように流れます。
* **音声の上書きについて**
  * 音声の上書きルールは以下のようになっています。
    * Not Loop属性のブレス：上書き不可。（初期設定ではリアクション時の単発ボイスなどが設定されています）
    * ボイス：Not Loopおよびボイスで上書き可能。
    * ブレス：すべての音声で上書き可能。
  * 上書きできない音声がカテゴリー変更などで流されようとした場合は、前回の音声の再生終了後に流れます。（音声再生を予約するイメージです）
  * ただし、Not Loopやボイスも音声再生の停止で等しく止まるため、タイムライン上で直前に 音声停止 → 再生 を行うことで、擬似的にブレス等でも上書きが可能です。
